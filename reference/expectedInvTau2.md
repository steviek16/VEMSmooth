# expectedInvTerms - Expected Values for Variance Parameters

Computes the expected inverse and logarithmic terms under the
Inverse-gamm distributions for or τ² and σ².

## Usage

``` r
expectedInvTau2(lambda1, lambda2)
```

## Arguments

- lambda1, lambda2:

  Shape and scale parameters for τ².

- delta1, delta2:

  Shape and scale parameters for σ².

## Value

Numeric expected values.
